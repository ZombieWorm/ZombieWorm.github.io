# Projects

A selection of my hands-on work in AI deployment, local LLMs, Python tooling, and home-lab experiments.

## Current / Highlighted Work

- **Local LLM Deployment Pipeline**  
  End-to-end setup for running quantized models (Llama 3, Mistral, Phi-3, etc.) on consumer hardware with GPU acceleration. Includes Ollama + Open WebUI + custom API wrappers.

- **Python Automation & Tooling Suite**  
  Collection of scripts for model conversion (gguf), benchmark automation, VRAM monitoring, and workflow orchestration. Heavily uses huggingface_hub, torch, and subprocess management.

- **Home-Lab Monitoring Dashboard**  
  Lightweight Grafana + Prometheus stack tuned for tracking GPU usage, power draw, temperature, and inference latency across multiple nodes.

- **Quantization & Optimization Experiments**  
  Comparative benchmarks of AWQ / GPTQ / bitsandbytes vs native FP16 on RTX 4090 / A6000-class cards. Focus on quality vs speed trade-offs.
